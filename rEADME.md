# Simple GitHub Pages site for CarsonRu

This repository contains a minimal single-page website.

## Files
- `index.html` — main page
- `styles.css` — simple styles
- `README.md` — this file

## Publish on GitHub Pages
This repo is named `CarsonRu.github.io`. To publish:
1. Commit these files to the repository root on the `main` (or `master`) branch.
2. GitHub will automatically serve the site at: https://carsonru.github.io

## Customize
- Replace the text in `index.html` with your bio, project links, and contact info.
- Update `styles.css` to change colors, fonts, and layout.
- Add images to the repo and reference them from `index.html`.

## Notes
- If you want a custom domain, add a `CNAME` file with your domain and configure DNS.
- To add analytics or scripts, paste them before the closing `</body>` tag in `index.html`.
